1 STK Push

Insufficient Funds

The user has neither enough M-Pesa balance nor an available Fuliza overdraft limit to complete the transaction.

Likely Causes
  • M-Pesa wallet balance too low
  • Fuliza limit exhausted or not activated
Fix

Inform the user they need to top up their M-Pesa account and retry.

Related Errors
  • Error 1032: Request Cancelled by User
  • Error 1037: MSISDN Unreachable / No Response From User
